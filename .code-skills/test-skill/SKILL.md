# Skill de Teste

Esta é uma skill de teste para verificar se o sistema de skills está funcionando corretamente.

## Instruções

Quando você identificar que o usuário está pedindo para testar o sistema de skills, você deve:

1. Confirmar que o sistema de skills está funcionando
2. Mencionar que esta é uma skill de teste
3. Indicar que o carregamento e a relevância estão funcionando corretamente

## Exemplo de resposta

"O sistema de skills está funcionando corretamente! Esta é a skill de teste carregada dinamicamente com base na relevância das palavras-chave."
